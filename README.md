# 🐍 Curso em Vídeo — Python (Gustavo Guanabara)

A repository containing all my **notes and exercises** from the Python course by **Gustavo Guanabara** on the [Curso em Vídeo](https://www.youtube.com/@CursoemVideo) YouTube channel — one of the most popular free Python courses in Brazil.

> ⚠️ This repository is constantly being updated as I progress through the course!

## 🎯 Goal

Store and organize every lesson and exercise from the course, documenting my learning journey through Python from the basics all the way to more advanced topics like functions, dictionaries, and exception handling.

## 📚 Concepts Covered So Far

- 🖨️ **Input & Output** — `input()`, `print()`, `.format()`, f-strings
- 🔀 **Conditionals** — `if`, `elif`, `else` and logical operators
- 🔁 **While loops** — counters, sentinels, `break` and `continue`
- 🔄 **For loops** — `range()`, `enumerate()`, `time.sleep()`
- 🔤 **String manipulation** — slicing, searching, replacing and formatting
- 🎲 **Random module** — generating random numbers for games and simulations
- 📋 **Lists & Tuples** — creation, sorting, inserting, slicing and searching
- 🔲 **Matrices** — nested lists and 2D data structures
- 📖 **Dictionaries** — key-value pairs, nested dicts, iteration
- 🧩 **Functions** — `def`, parameters, `*args`, `return`, scope, default values
- 🛡️ **Exception Handling** — `try/except/else/finally`, error types, `urllib`
- 🎨 **Terminal formatting** — ANSI color codes for styled output
